# HTTrack Website Copier - GitHub Actions

פרויקט זה מאפשר להוריד אתרים באמצעות HTTrack דרך GitHub Actions.

## שימוש

### הרצה דרך GitHub Actions

1. עבור ללשונית **Actions** בריפוזיטורי שלך
2. בחר ב-workflow **"HTTrack Website Copier"**
3. לחץ על **"Run workflow"**
4. הזן את הפרטים:
   - **Website URL**: הכנס את כתובת האתר (לדוגמה: `https://example.com`)
   - **Project name**: שם לפרויקט (אופציונלי, ברירת מחדל: "website")
5. לחץ על **"Run workflow"** הירוק

### הורדת התוצאות

לאחר שה-workflow מסתיים:
1. היכנס לדף ה-workflow run
2. גלול למטה ל-**Artifacts**
3. הורד את הקובץ הדחוס עם האתר שהורד

## הגדרות HTTrack

ה-workflow משתמש בהגדרות הבאות:
- `-v`: מצב verbose (מידע מפורט)
- `-%v`: הצג התקדמות
- `-N0`: אל תשמור קבצים זמניים
- `-s0`: אל תעקוב אחרי קישורים חיצוניים
- `-r6`: עומק רקורסיה מקסימלי של 6 רמות
- `-%e0`: אל תשתמש ב-cache חיצוני

## דרישות

- ריפוזיטורי GitHub (פרטי או ציבורי)
- הרשאות GitHub Actions מופעלות

## הערות

- הקבצים שמורדים נשמרים ל-7 ימים (ניתן לשנות ב-`retention-days`)
- זמן ההרצה מוגבל לפי מגבלות GitHub Actions (בדרך כלל 6 שעות)
- שים לב לכבד את תנאי השימוש של האתרים שאתה מוריד
